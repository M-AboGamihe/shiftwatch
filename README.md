### AI-powered Workplace Monitoring & Employee Tracking System

---

## 📌 Overview

ShiftWatch is an intelligent Flutter application designed to help organizations monitor employee activity, manage shifts, and analyze workplace behavior using AI-powered video processing.

The system transforms surveillance videos into actionable insights such as attendance tracking, movement detection, and working hours calculation.

---

## 🧠 Project Idea

The supervisor uploads a workplace surveillance video, and the system automatically:

- 📅 Detects employee attendance times  
- 🚶 Tracks movement inside designated work zones  
- ⏱ Calculates actual working hours  
- 📊 Generates analytics and performance insights

# 🚀 Features

- 🔐 Firebase Authentication (Login / Register)
- 🎥 AI-based Video Analysis
- 📍 Polygon-based Work Zone Mapping
- 👨‍🏭 Employee Management System
- 📊 Real-time Analytics Dashboard
- ☁️ Firebase + Azure Cloud Integration
- 🛎 Firebase Cloud Messaging (Push Notifications)
- 🌍 Multi-language Support (EN / AR)
- ⚙️ Full Profile & Settings Management
- 🧠 Clean Architecture + BLoC State Management

---

# 🧱 Architecture

This project is fully refactored using modern Flutter architecture:

- 🧠 Clean Architecture
- 🔄 BLoC (Business Logic Component)
- 📦 Feature-based structure
- 🧩 Scalable modular design
